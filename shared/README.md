# Impulse Purchase Predictor

A cross-platform browser extension and AI backend to predict and prevent impulse purchases.

## Structure
- `extension/`: Browser extension (Node.js, JS/TS)
- `backend/`: Node.js backend server
- `ai/`: Python AI/ML analysis
- `shared/`: Shared schemas, docs, configs

## How it works
1. Extension collects browsing and behavioral data.
2. Sends data to backend API.
3. Backend forwards to Python AI for prediction.
4. Extension displays real-time warnings and advice.
